# Dynamic AOP

using Runtime machanism to monitor method invoke, and monitor configuration can be configed by server side

- Real dynamic AOP
- You can config ClassName & MethodName in server side, and iOS side can monitor this method automatically
- iOS side can statistics arguments and return value, no matter how much the arguments is in method
- the argument type of method support: int, long, bool, double, class and block
- the return type of method support: int, long, bool, double, class, void
